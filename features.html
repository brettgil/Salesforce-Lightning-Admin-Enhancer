Salesforce Lightning Admin Enhancer

A suite of productivity enhancements for Salesforce Lightning admins. All features are optional and can be toggled individually.

Lightning
Enhancements to the core Salesforce Lightning experience — the header, navigation, and record pages you use every day.
⚡
Clickable Logo
The Salesforce logo in the Lightning header does nothing by default. This feature makes it clickable — opening your Lightning home page in a new tab. A small change that saves time when you need to quickly jump back to home without losing what you're working on.
🔗
Launch by ID
Adds an input field next to the Lightning search bar. Paste any 15 or 18-digit Salesforce record ID, a relative path, or a full Salesforce URL and press Enter to navigate directly. When a full URL is pasted, the hostname is stripped so it opens in your current org — useful when moving from production to sandbox for the same record.
Works with bare IDs, relative paths, and full URLs from any org
↩️
App Switcher Return
When you switch apps using the Salesforce App Launcher, you're dropped on the new app's home page and lose your place. This feature remembers the record you were viewing and either automatically redirects you back, or shows a dismissible banner with a link.
Choose auto-redirect or show prompt from Settings
👤
Login As Return
When an admin logs out of a Login As session, Salesforce returns them to the Users list instead of the user they were just viewing. This feature automatically redirects back to that user's detail page in Setup after logging out.
🪪
Record ID Display
Shows the current record ID in the Lightning header on any record page. Click it to copy to clipboard. Updates automatically as you navigate between records.
Handy for sharing IDs with colleagues or pasting into support tickets
🏷️
Field API Name
Right-click any field on a Lightning record page to reveal its API name directly below the field label. Right-click again to hide it. No popups, no navigation — the API name appears inline exactly where you need it.
Useful for formulas, SOQL queries, and debugging field mappings
🔍
User Search
Adds a user search button directly to the search bar. Search for any user by name, username, or email — including deactivated users that Salesforce's built-in search ignores. Results show name, username, and an Active or Inactive badge. Choose whether clicking a result takes you to the Setup user page or the Lightning record page.
Available in both Lightning and Setup
Setup
Quality-of-life improvements for Salesforce Setup — faster navigation, smarter Quick Find, and shortcuts that stick.
🏢
Org ID Display
Displays your organization's 18-digit Org ID in the Setup header — click to copy. On Setup pages that show a specific record (such as a user, permission set, or channel), the header automatically switches to display that Record ID instead. Falls back to the Org ID on pages with no record context.
Org ID on the homepage, Record ID everywhere else it applies
🔎
Quick Find Layout
In the Object Manager, the Quick Find filter is positioned on the right side of the header — away from the list you're filtering. This feature moves it to the left, keeping it visually aligned with the content below. Also applies to Fields & Relationships and Page Layouts.
⌨️
Quick Focus
Automatically moves focus to the Quick Find input when Setup pages load. Start typing your search term the moment the page is ready — no need to click the field first. Small, but makes navigating Setup noticeably faster.
⭐
Setup Favorites
Salesforce's native Favorites feature isn't available in Setup by default. This feature adds the star icon and favorites dropdown to Setup pages — the same experience you get in Lightning. Your favorites sync with your Salesforce account.
📌
My Setup Shortcuts
Adds a configurable shortcuts section to the Setup sidebar. Define your own list of frequently-used Setup pages — labels and URLs — and access them with one click. Supports drag-to-reorder and a quick-pin button that appears when hovering over any Setup sidebar item.
Managed from the Settings page
Process Builder
Enhancements to the Salesforce Process Builder interface to make it easier to read, resize, and work with field values.
↔️
Drag to Resize Panel
Adds a drag handle to the criteria and action panel so you can resize it horizontally. Useful when working with wide field values or long condition lists that get clipped in the default layout.
📋
Copy Text Buttons
Adds a one-click Copy button below text fields in the action and outcome panels. Grab field values instantly without having to select and copy text manually.
🏷️
Full API Names in Modals
Field selection modals in Process Builder truncate long API names, making it hard to distinguish between similar fields. This feature expands the name display so the full API name is always visible.
Snippets
Store frequently used text — formulas, SOQL queries, notes — and copy them instantly from anywhere.
📋
Snippets
Store frequently used formulas, SOQL queries, and notes for quick access. Snippets are managed from the Settings page — add a label and content, then click Copy to instantly copy the full text to your clipboard. Snippets save immediately and support drag-to-reorder.
Perfect for SOQL queries, formula fields, and boilerplate text you reuse often
