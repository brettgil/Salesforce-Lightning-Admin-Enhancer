Toggle features on or off. Save and reload your Salesforce page to apply changes.

Feature Tour → What's New →
Lightning
Clickable Logo
Makes the Salesforce logo clickable — opens Lightning home in a new tab.
Launch by ID
Adds an input next to the search bar — paste a record ID, a relative path, or a full URL to navigate directly. Full URLs are stripped to the current org, useful when moving from production to sandbox for the same record.
App Switcher Return
When switching apps, return to the record you were viewing.
Login As Return
After logging out of a Login As session, automatically redirect to that user's detail page.
Record ID Display
Shows the current record ID in the header on any record page — click to copy.
Field API Name
Right-click any field on a Lightning record page to show its API name below the field label. Right-click again to hide it.
User Search
Adds a user icon to the header. Search any user by email or username — including inactive users — and navigate directly to their record.
Click destination
Where to navigate when clicking a user result.
Object Manager — Load More
Loads 500 records at a time when browsing a specific object's lists in Object Manager — Fields, Validation Rules, Page Layouts, and more. Scrolling to the bottom loads the next 500 automatically instead of the default 50.
Setup
Org ID Display
Shows the Org ID in the Setup header — click to copy. Switches to Record ID on Setup pages with a specific record.
Quick Find Layout
Moves the Quick Find filter to the left side of the Object Manager header.
Quick Focus
Auto-focuses the Quick Find input when Setup pages load.
Setup Favorites
Enables the Favorites star and dropdown in Setup, synced with your Salesforce favorites.
My Setup Shortcuts
Adds a custom shortcuts section to the Setup sidebar.
Process Builder
Drag-to-resize panel, copy text buttons, full API names in modals.
